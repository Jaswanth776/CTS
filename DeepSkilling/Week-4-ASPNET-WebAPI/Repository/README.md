# Repository Pattern

This folder contains the Repository Pattern implementation used to abstract data access logic.

## Topics Covered

- Repository Pattern
- Interfaces
- Dependency Injection
- CRUD Operations

## Files

### IEmployeeRepository.cs

Defines operations for employee data access.

### EmployeeRepository.cs

Implements employee data access methods.

## Learning Outcomes

- Understand the Repository Pattern.
- Separate data access from business logic.
- Improve application maintainability.
- Support dependency injection.
