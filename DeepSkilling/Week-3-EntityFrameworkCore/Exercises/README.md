# Exercises

This folder contains practical exercises completed during the Entity Framework Core module.

## Exercises

### Exercise 1 - Employee Entity Creation
- Create an Employee model.
- Define properties and primary key.

### Exercise 2 - DbContext Configuration
- Configure ApplicationDbContext.
- Add DbSet<Employee>.

### Exercise 3 - CRUD Operations
- Insert employee records.
- Retrieve employee records.
- Update employee details.
- Delete employee records.

### Exercise 4 - LINQ Queries
- Filter records.
- Sort records.
- Perform aggregate operations.

## Learning Outcomes

- Apply Entity Framework Core concepts.
- Build database-driven applications.
- Practice CRUD and LINQ operations.
