# LINQ Queries

This folder contains examples demonstrating Language Integrated Query (LINQ) with Entity Framework Core.

## Topics Covered

- Filtering
- Sorting
- Projection
- Aggregation

## Files

### LINQQueries.cs

Demonstrates common LINQ operations on Employee entities.

## Learning Outcomes

- Retrieve filtered records.
- Sort query results.
- Select specific properties.
- Perform aggregate calculations.
- Use LINQ with Entity Framework Core.
