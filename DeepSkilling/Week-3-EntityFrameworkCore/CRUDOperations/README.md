# CRUD Operations

This folder contains examples demonstrating Create, Read, Update, and Delete operations using Entity Framework Core.

## Topics Covered

- Create
- Read
- Update
- Delete
- SaveChanges()

## Files

### CRUDExample.cs

Demonstrates basic CRUD operations on the Employee entity.

## Learning Outcomes

- Insert records into the database.
- Retrieve records from the database.
- Modify existing records.
- Delete records from the database.
- Persist changes using SaveChanges().
