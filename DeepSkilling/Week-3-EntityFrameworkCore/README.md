# Week-3 Entity Framework Core

This module contains hands-on practice and examples related to Entity Framework Core as part of the Cognizant Digital Nurture 5.0 (DN 5.0) DeepSkilling program.

## Topics Covered

### Models

* Entity Classes
* Primary Keys
* Properties and Data Types

### DbContext

* Creating DbContext Classes
* Database Configuration
* DbSet Usage

### CRUD Operations

* Create Records
* Read Records
* Update Records
* Delete Records

### LINQ Queries

* Filtering Data
* Sorting Records
* Projection
* Aggregation

### Migrations

* Creating Migrations
* Updating Databases
* Managing Schema Changes

### Exercises

* Practical Entity Framework Core Exercises
* Database Interaction Examples

## Learning Outcomes

* Understand Entity Framework Core architecture.
* Create and manage entity models.
* Configure and use DbContext.
* Perform CRUD operations using EF Core.
* Execute LINQ queries on database entities.
* Manage database schema using migrations.
* Build database-driven applications using C# and EF Core.
