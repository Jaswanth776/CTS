# Entity Framework Core Quiz Notes

This folder contains notes and important concepts learned during the Entity Framework Core module.

## Important Concepts

### Entity Framework Core
- ORM Framework for .NET
- Maps C# classes to database tables

### DbContext
- Central class for database communication
- Tracks entity changes

### DbSet
- Represents a database table

### Migrations
- Manage database schema changes
- Keep database synchronized with models

### CRUD Operations
- Create
- Read
- Update
- Delete

### LINQ
- Query data using C# syntax
- Supports filtering, sorting, projection, and aggregation

## Learning Outcomes

- Understand EF Core architecture.
- Work with entities and DbContext.
- Perform CRUD operations.
- Use LINQ with Entity Framework Core.
- Manage schema changes using migrations.
