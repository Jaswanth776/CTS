# Performance

This folder contains examples of performance optimization techniques in Entity Framework Core.

## Topics Covered

- AsNoTracking
- Compiled Queries
- Bulk Operations

## Files

### AsNoTrackingExample.cs
Demonstrates read-only queries using AsNoTracking().

### CompiledQueryExample.cs
Demonstrates compiled queries for better performance.

## Learning Outcomes

- Improve query performance.
- Use AsNoTracking for read-only data.
- Implement compiled queries.
- Understand bulk operations.
