# Concurrency

This folder contains examples of handling concurrent updates in Entity Framework Core.

## Topics Covered

- Optimistic Concurrency
- RowVersion
- Concurrency Tokens

## Files

### RowVersionExample.cs

Demonstrates handling concurrent updates using RowVersion.

## Learning Outcomes

- Understand concurrency conflicts.
- Implement optimistic concurrency control.
- Use RowVersion for change tracking.
- Handle simultaneous database updates safely.
