# Models

This folder contains entity classes used by Entity Framework Core.

## Topics Covered

* Entity Classes
* Properties and Data Types
* Primary Keys
* Database Table Mapping

## Files

### Employee.cs

Represents an Employee entity that will be mapped to a database table using Entity Framework Core.

## Learning Outcomes

* Create entity classes.
* Define properties and data types.
* Understand how entities are mapped to database tables.
* Use primary keys for entity identification.
