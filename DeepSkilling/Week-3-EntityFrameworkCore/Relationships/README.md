# Relationships

This folder contains examples of relationships in Entity Framework Core.

## Topics Covered

- One-to-One Relationships
- Many-to-Many Relationships
- Navigation Properties
- Foreign Keys

## Files

### ProductDetail.cs
Demonstrates a one-to-one relationship.

### Tag.cs
Demonstrates a many-to-many relationship.

## Learning Outcomes

- Configure one-to-one relationships.
- Configure many-to-many relationships.
- Use navigation properties.
- Work with foreign keys.
