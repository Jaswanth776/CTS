# Migrations

This folder contains notes related to Entity Framework Core migrations.

## Topics Covered

- Initial Migration
- Updating Database
- Schema Changes
- Migration Commands

## Common Commands

### Create Migration

dotnet ef migrations add InitialCreate

### Update Database

dotnet ef database update

### Remove Migration

dotnet ef migrations remove

## Learning Outcomes

- Understand EF Core migrations.
- Manage database schema changes.
- Apply migrations to databases.
- Maintain version-controlled database structures.
