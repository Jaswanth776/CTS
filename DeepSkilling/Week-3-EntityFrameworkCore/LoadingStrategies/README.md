# Loading Strategies

This folder contains examples of loading related data in Entity Framework Core.

## Topics Covered

- Eager Loading
- Explicit Loading
- Lazy Loading

## Files

### EagerLoading.cs
Demonstrates loading related entities using Include().

### ExplicitLoading.cs
Demonstrates loading related entities manually.

## Learning Outcomes

- Understand loading strategies.
- Load related entities efficiently.
- Improve query performance.
