# Week 1 - Design Patterns and Principles

This folder contains implementations of software design principles and design patterns as part of the Cognizant Digital Nurture 5.0 Deep Skilling program.

## SOLID Principles

- Single Responsibility Principle (SRP)
- Open/Closed Principle (OCP)
- Liskov Substitution Principle (LSP)
- Interface Segregation Principle (ISP)
- Dependency Inversion Principle (DIP)

## Design Patterns

### Creational Patterns
- Singleton Pattern
- Factory Method Pattern

### Structural Patterns
- Adapter Pattern
- Decorator Pattern

### Behavioral Patterns
- Strategy Pattern
- Observer Pattern

## Objective

To understand how maintainable, reusable, and scalable software systems are designed using established software engineering principles.
