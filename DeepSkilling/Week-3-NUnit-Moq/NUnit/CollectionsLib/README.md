# Collection Tests

This exercise demonstrates CollectionAssert methods in NUnit.

## Topics Covered

- CollectionAssert.Contains
- CollectionAssert.AllItemsAreUnique
- CollectionAssert.AllItemsAreNotNull

## Files

### Employee.cs
Represents employee data.

### EmployeeCollectionTests.cs
Tests employee collections using CollectionAssert.

## Learning Outcomes

- Validate collection contents.
- Check uniqueness of values.
- Ensure collections contain valid data.
- Use CollectionAssert effectively.
