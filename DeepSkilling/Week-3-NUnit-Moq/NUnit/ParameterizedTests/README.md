# Parameterized Tests

This exercise demonstrates NUnit parameterized testing using TestCase attributes.

## Topics Covered

- TestCase
- Assertions
- Exception Testing
- Multiple Test Inputs

## Files

### Calculator.cs
Calculator implementation for arithmetic operations.

### ParameterizedCalculatorTests.cs
Parameterized NUnit tests for Add, Subtract, Multiply and Divide methods.

## Learning Outcomes

- Execute tests with multiple input values.
- Reduce code duplication using TestCase.
- Validate exception handling.
- Improve test coverage efficiently.
