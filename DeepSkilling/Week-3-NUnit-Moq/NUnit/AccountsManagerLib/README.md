# Account Manager Tests

This exercise demonstrates login validation testing using NUnit.

## Topics Covered

- Boolean Assertions
- Exception Testing
- Login Validation
- Business Logic Testing

## Files

### AccountManager.cs

Simple login validation class.

### AccountManagerTests.cs

Tests valid login, invalid login, and exception scenarios.

## Learning Outcomes

- Test authentication logic.
- Verify expected boolean results.
- Validate exception handling.
- Improve reliability of business logic.
