# Leap Year Tests

This exercise demonstrates NUnit testing using TestCase attributes.

## Topics Covered

- TestCase
- Boolean Assertions
- Date Validation

## Files

### LeapYearChecker.cs
Checks whether a given year is a leap year.

### LeapYearTests.cs
Contains NUnit tests for leap year validation.

## Learning Outcomes

- Validate business rules using unit tests.
- Use parameterized testing.
- Improve code reliability.
