# TestCaseSource Tests

This exercise demonstrates supplying test data using TestCaseSource.

## Topics Covered

- TestCaseSource
- External Test Data
- Parameterized Tests

## Files

### Season.cs
Provides season logic and test data source.

### SeasonTests.cs
Uses TestCaseSource to execute tests with external data.

## Learning Outcomes

- Create reusable test data.
- Separate test logic from test data.
- Improve maintainability of unit tests.
