# Calculator Tests

This exercise demonstrates basic NUnit testing using TestFixture, SetUp, TearDown, and Assert methods.

## Topics Covered

- TestFixture
- Test Method
- SetUp
- TearDown
- Assert.AreEqual

## Files

### Calculator.cs

Simple calculator class containing an Add method.

### CalculatorTests.cs

Unit test verifying addition functionality.

## Learning Outcomes

- Create NUnit test classes.
- Write test methods.
- Verify expected results using assertions.
- Initialize and clean up test resources.
