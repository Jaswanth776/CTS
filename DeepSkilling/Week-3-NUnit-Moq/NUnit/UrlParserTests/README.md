# URL Parser Tests

This exercise demonstrates NUnit assertions using Assert.That.

## Topics Covered

- Assert.That
- String Validation
- URL Parsing
- Unit Testing

## Files

### UrlParser.cs
Extracts the hostname from a URL.

### UrlParserTests.cs
Verifies hostname extraction using NUnit.

## Learning Outcomes

- Understand Assert.That syntax.
- Test string-based functionality.
- Validate URL parsing logic.
