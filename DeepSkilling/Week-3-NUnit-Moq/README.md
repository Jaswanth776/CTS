# Week-3 NUnit and Moq

This module contains hands-on exercises, unit testing examples, and mock object implementations using NUnit and Moq as part of the Cognizant Digital Nurture 5.0 (DN 5.0) DeepSkilling program.

## Module Overview

### NUnit

NUnit is a popular unit testing framework for .NET applications. It is used to create automated tests that verify the correctness of application logic.

Topics covered:

- Test Fixtures
- Assertions
- Test Cases
- Parameterized Tests
- Collection Assertions
- Exception Testing
- TestCaseSource
- Setup and Teardown Methods

### Moq

Moq is a mocking framework used to isolate dependencies while testing applications.

Topics covered:

- Mock Objects
- Dependency Injection Testing
- Verifying Method Calls
- Mocking External Services
- Mocking File Systems
- Mocking Database Layers
