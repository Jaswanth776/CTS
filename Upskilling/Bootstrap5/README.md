# Bootstrap5

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. grid_system.html

   * Demonstrates Bootstrap containers, rows, columns, and responsive layouts.

2. navbar_demo.html

   * Demonstrates responsive navigation bars and Bootstrap navigation components.

3. cards_demo.html

   * Demonstrates Bootstrap cards, shadows, and utility classes.

4. forms_demo.html

   * Demonstrates Bootstrap forms, input groups, and form controls.

5. typography_demo.html

   * Demonstrates Bootstrap typography classes and text utilities.

6. buttons_demo.html

   * Demonstrates Bootstrap button styles, sizes, and states.

7. spacing_demo.html

   * Demonstrates Bootstrap spacing utilities including margin and padding.

8. flexbox_demo.html

   * Demonstrates Bootstrap flexbox utilities and alignment classes.

9. modal_demo.html

   * Demonstrates Bootstrap modal components.

10. accordion_demo.html

    * Demonstrates Bootstrap accordion and collapse components.

## Learning Outcomes

* Understand Bootstrap grid systems
* Create responsive layouts
* Build navigation bars and cards
* Design professional forms
* Apply Bootstrap utility classes
* Use Bootstrap typography features
* Implement spacing and flexbox utilities
* Create interactive modals and accordions
